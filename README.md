# 🫐 Elder-Berry

> Virtual AI Assistant with V-Tuber character, sensor integration and robot control.
> Part of the Last-Strawberry project family.

## Overview
Elder-Berry is a modular AI assistant system built around a V-Tuber character interface.
It combines local LLM processing, sensor data and PC control into a single cohesive assistant.

## Hardware
| Component | Role |
|---|---|
| Tower PC (RTX 4070 Ti Super, 16GB VRAM) | LLM Host, main processing |
| Raspberry Pi 5 | I/O Controller (sensors, motors) |
| Mecanum Robot | Mobile carrier, autonomous charging |

## LLM Strategy
| Mode | Model | Use Case |
|---|---|---|
| Local | Ollama 14B | Fast actions, PC control, sensors |
| Remote | OpenRouter | Multimodal, complex reasoning |
